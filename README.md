# FizzBuzz-Programming-Task

Write a Python program that:

Asks the user to enter 5 numbers, one at a time.

For each number:

Print "FizzBuzz" if divisible by both 3 and 5

Print "Fizz" if divisible by only 3

Print "Buzz" if divisible by only 5

Otherwise, print the number itself

If the user enters something not a number, display "Enter a number" and let the user re-enter the number.
